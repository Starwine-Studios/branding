<h3 id="rtf-extension---is-a-resource-pack-that-improves-and-expands-russian-translations-for-minecraft-mods-on-versions-1165-to-121">RTF Extension - is a resource pack that improves and expands Russian translations for Minecraft mods on versions 1.16.5 to 1.21+</h3>
<p>It replace the original RTF project, which is no longer maintained.</p>
<h2 id="note">Note</h2>
<p>All new translations are created and tested on Minecraft 1.20.1 and above. Some translations may not work correctly on older versions.</p>
